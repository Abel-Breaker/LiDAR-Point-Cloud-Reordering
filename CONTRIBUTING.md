# How to Contribute

Thank you for your interest in contributing to this project! This document explains how to collaborate effectively and ensures that all contributions maintain code quality.


## Coding Guidelines

To maintain project quality, all contributions must adhere to the following:

- Code must be **well-formatted** using the provided `.clang-format` file, which follows a style similar to the **Linux Kernel Style** but with minor modifications.

- Maintain consistency in variable names, comments, and overall style.

- Code must **compile without errors or warnings** using `make MODE=debug` and `make MODE=debug_threads`
