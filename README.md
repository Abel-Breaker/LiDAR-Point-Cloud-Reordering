El principal objetivo de este Trabajo de Fin de Grado es reordenar una nube de puntos LiDAR a partir de las relaciones de vecindad entre sus puntos dando lugar a una matriz de vecindad simétrica dispersa a la que se aplicarán algoritmos de reordenación para reducir su ancho de banda o para obtener particiones por bloques. ALgunas alternativas que se valorarán son: Cuthill–McKee, Minimum Degree, Nested Dissection o Cuthill–McKee Inverso. Mediante estas estrategias, se busca mejorar la localidad espacial de los datos con el fin de optimizar la eficiencia en su acceso a memoria en su procesamiento posterior. Se estudiará en detalle su rendimiento y escalabilidad.
